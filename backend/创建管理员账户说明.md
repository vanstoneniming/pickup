# 创建管理员账户说明

## 方法一：使用 Node.js 脚本（推荐）

### 步骤 1：确保数据库已创建

```bash
cd backend
./init-database.sh
```

### 步骤 2：创建管理员账户

```bash
node create-admin.js
```

这会创建一个默认管理员账户：
- **用户名**: `admin`
- **密码**: `admin123`
- **角色**: `admin`

### 步骤 3：登录系统

使用以下信息登录：
- 用户名：`admin`
- 密码：`admin123`

⚠️ **重要**：登录后请立即修改密码！

## 方法二：使用注册接口

启动后端服务后，使用注册接口创建管理员：

```bash
curl -X POST http://localhost:3000/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "admin123",
    "real_name": "系统管理员"
  }'
```

然后手动在数据库中修改角色为 admin：

```sql
USE pickup_db;
UPDATE users SET role = 'admin' WHERE username = 'admin';
```

## 方法三：直接使用 SQL（不推荐）

如果知道 bcrypt 加密后的密码，可以直接插入：

```sql
USE pickup_db;

-- 密码 "admin123" 的 bcrypt hash
INSERT INTO users (username, password, role, real_name, status) 
VALUES ('admin', '$2a$10$rK8Z5Z5Z5Z5Z5Z5Z5Z5Z5uK8Z5Z5Z5Z5Z5Z5Z5Z5Z5Z5Z5Z5Z5Z5Z', 'admin', '系统管理员', 1);
```

## 默认账户信息总结

### 系统登录账户
- **用户名**: `admin`
- **密码**: `admin123`（首次登录后需修改）
- **角色**: 管理员

### 数据库连接账户
- **用户名**: `pickup_db`
- **密码**: `pickup_db`
- **数据库**: `pickup_db`

## 注意事项

1. 默认密码仅用于首次登录，请立即修改
2. 生产环境务必使用强密码
3. 建议为不同管理员创建不同的账户
